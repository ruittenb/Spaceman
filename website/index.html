Spaceman is an application for macOS that allows you to view your Spaces (Virtual Desktops) in the menu bar. Spaceman allows you to see which space you are currently on (or spaces if you are using multiple displays) relative to the other spaces you have. Naming these spaces is also an option in order to organise separate spaces for your workflow.

About This Fork

This is an actively maintained fork of the original Spaceman application created by Sasindu Jayasinghe. We appreciate this useful app and all the foundational work that went into it. You can visit the original project page and original repository.

Unfortunately, the main repository is no longer actively maintained. This fork continues development with many new features, bug fixes, and improvements to keep Spaceman working with the latest macOS versions.

  • Customizable

    Spaceman can be customized with 5 different icon styles in 5 different sizes. Have a minimal view of your space or have them named.

  • Interactive

    Click directly on the menu bar icon to instantly switch between your spaces. Navigate between virtual desktops with a simple click.

  • Native

    Written in Swift and Objective-C and compiled to a universal binary to support both Apple Silicon and Intel Macs.

  • Open-source

    All code used to build this application is openly available on GitHub with the MIT license. Pull requests are welcome.
